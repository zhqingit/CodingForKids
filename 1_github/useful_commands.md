1. clone repository from server to local 
    ```
    git clone  git@github.com:zhqingit/test.git
    ```
2. add files
    ```
    git add file
    ```
    or 
    ```
    git add .
    ```
3. check the status
    ```
    git status
    ```
4. commit the change
    ```
    git commit -m "add files"
    ```

5. Push your code to github
    ```
    git push -u origin main
    ```

6. create new branch
    ```
    git checkout -b new_branch
    ```
